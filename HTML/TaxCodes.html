Tax Codes






  • Tax Codes can be used to define specific tax rates and can be accessed from the Main Menu of the Home screen. 
  • User's will have the option to Create, Edit or Delete a record from this screen.
  • To create a new Tax Code click the button at the top of the screen.

Add Tax Codes



  • From the Create Tax Code screen, enter your Tax Code details.
  • Click the if you would like to add any additional tax codes.




  • Enter a Description, set the tax percentage and set a Start and End Date. 



  • To delete a Tax Code within the Edit Tax Code screen, click the button for the Tax Code you would like to remove. 
  • You will not see a confirmation screen, the item will just be removed.




  • NOTE: You cannot delete a Tax Code if it has been used in a Price Sheet on a Sales Order that has been used on a ticket.









Edit a Tax Code 



  • From the Tax Code main screen, click on the button to open the Edit Tax Code screen.  
  • You cannot update an existing Tax Code, only remove/add a new item.






Delete a Tax Code 



  • To delete a Tax Code from the main Tax Code screen, click the button for the Tax Code you would like to remove. 




  • On the confirmation screen, click the button to confirm you would like to delete the Tax Code.
  • Note: You cannot delete a Tax Code if it has been used in a Price Sheet or a Sales Order.
