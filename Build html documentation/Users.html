Users






Create New User



To create a new user, click the 'New User' button located inside the User settings box. 




From the User Administration main screen, click the 'Create New User' button.

Enter the User's information and select a role and permission set for that user. 

When finished, click the 'Create' button. The user will now show on the User Administration screen.

Email User


Clicking on a User's email address will open your default email program with User's email address in to mailto field.




Edit User


To Edit a User, click on the Edit link to the right for the User you would like to make changes to.

 If User Role is set to Admin, this Permission Set field is grayed out (because an Admin has the highest set of permission).




 From the Manage Users main screen, you will be able to modified any field.

When finished editing, click the button or if you do not want to save the changes you made.






Reset Password


If you would like to reset your password, click on for the User you would like to reset the password for.





Enter the new password you would like to have set and click .




A confirmation screen will appear saying the password has been reset. 

Passwords must be at least 10 characters in length.




Delete a User


To Delete a User, click on the Delete button to the right for the Hauler you would like to remove.




You will be prompted to confirm the deletion. Reaffirm by clicking the button. 

You will then be returned to the User Administration page and the User has been removed.






Locked Out Users



If a user has had too many unsuccessful attempts at logging in, they will be locked out.




To unlock a user, you must be logged in as a user that have admin permission. 

Open the Users menu under System Management and click the Edit button for the locked out user. 

Remove the check from the Locked Out field at the top of the page and click the Save button. 

The user should now be able to log into Fulcrum.
