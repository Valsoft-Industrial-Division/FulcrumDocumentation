Manage Permission Sets





  • From this screen you can Search, Update, or remove current a permission set. The Weighmaster Permission Set is in the list by default.







Create Permission Set



  • Set a name for your new permission set and click on .
  • You will be returned to the main Permission set screen and your new permission set will show in the list.






Update Permission Set



  • From here, you can click on the button to edit your Permission Set. 
  • When finished editing, click the button, or if you do not want to save the changes you made.









Delete Permission Set

  • To delete a permission set, click the button on the permission set you would like to remove. 




  • You will be prompted to confirm the deletion. Reaffirm by clicking the button. 
  • You will then be returned to the main permissions set screen and the permission set has been removed.
