Icon Guide


Fulcrum  Menu's



  • Fulcrum has a static navigation bar at the top that can be accessed from any menu.

Provides access to Ticket List, Create Ticket, Vehicles, Trailers, Haulers, Zones, Customers, Sales Orders, Product Classes, Products, Locations, Units of Measure, Rate Types, Tax Codes, Price Sheets

Takes you to the Report Menu and shows list of Reports.

Opens the Export Menu and shows a list of available Exports.

Provides access to Quickbooks Online and Escrow Payments.

Shows any active User tables that have been created.

Allows customization of site configuration and setup/maintenance of users.







Fulcrum Standard Icons



  • There are a few standard functions that you will find in all Fulcrum menu's.

Full text search that allows users to search active records in your current menu.

Allows you to select how many records are displayed per page. Users can pick 10, 25, 50, or 100 entries.

Appears at the bottom of each menu grid. Shows current records being displayed and the total number entries. 

Lets you navigate to previous pages or ahead pages in your current Menu.

Opens a new window for user to create a new menu item. 

Once a new item has been entered, the Create button creates the new record and takes you back to the main menu. The Back to List button will take you back to the main menu without saving your new entry. 

When editing a record, click the Save button to save changes and return to the main menu. Clicking the Back to List button will return you to the main menu without saving your changes.

From the main menu screen, each entry has the option to Edit or Delete the record.

The calendar icon appears on fields that require a date entry. Clicking the calendar will pop up a visual calendar that you can select your preferred date.

Most menu columns have a sort filter (up/down arrows) that can be used to sort the column in ascending or descending order.

Used to select the start date for running Reports, creating Sales Orders, etc.

Used to select the end date for running Reports, creating Sales Orders, etc
