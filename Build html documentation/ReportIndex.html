Report Index






Under the Reports Menu, you can access a list of available reports (available reports shown are based on user permissions set by an administrator). 

Each report will have its own set of parameters to display data specific to the report. 

Stock Reports are standard reports and cannot be written over.

The Reports that are Ticket Forms are not intended to be run as Reports, they are just the Report that prints when you print a ticket.








Select Report Criteria


All Reports will contain two static fields (FromDate and ToDate) to select date range criteria for your Report.





Filtering fields will vary based on the selected Report's criteria.

Each Report will have a button that displays your Report based on the criteria you've selected.

The returns you to the main Report Menu.



From the Report screen, select a date range to run your report. Once you've selected the From Date and To Date, click on Display Report.








Displayed Report



Each Report will contain a standard toolbar allowing you to Refresh the page, Print your Report, Zoom In/Out and navigate pages 

of the Report.

The left side navigation menu allows you to drill down on each Customers Sales Orders and Tickets.





Add Custom Report



You can create a Custom report by clicking the 'Add New Report' button at the top of the Report List screen.




Click the button to open file explorer and select your custom .frx file.

Give the Report a name and check the Ticket Form box if applicable. 

Unlike Stock Reports, Custom Reports can be deleted. 




You will now see your new report show in the Report List. 

Since it is a custom report, you will have the ability to upload a new or updated .frx file for the report also have the ability to delete the report.
