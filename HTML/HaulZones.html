Haul Zones








  • The Zones setup can be accessed from the Main Menu of the Home screen. 
  • User's will have the option to Create, Edit or Delete a record from this screen.






Create new Haul Zone




  • Click the button to add a new Haul Zone.


Description - Enter a description for your Zone.
External ID - Can to used to add a unique value or make reference to data you are importing from another system.
Inactive - Mark if a Zone is Active or Inactive.
Miles - Determines the boundaries of the Zone in miles.






Edit Haul Zone



  • To Edit a Haul Zone, click on the Edit link to the right for the Zone you would like to make changes to.



  • From the Edit Haul Zone screen, you will be able to modified any field. 
  • When finished editing, click or Cancel if you do not want to save the changes you made. 





Delete Zones



  • To Delete a Zone, click on the Delete button to the right for the Zone you would like to remove.




  • You will be prompted to confirm the deletion. Reaffirm by clicking the 'Delete' button. 
  • You will then be returned to the main Zones List screen and the Zone has been removed.
