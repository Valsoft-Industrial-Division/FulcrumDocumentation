Manage Site






You can create a new site, edit your site information  or search active sites from this menu.

Tickets and generated Reports will show this site information. 






Create New Site



To create a new site, click the Create New button at the top of the screen. 




Description

Site Description.

External ID

Can to used to add a unique value or make reference to data you are importing from another system.

Inactive

Mark site as Active/Inactive.

Ticket Prefix

If you have multiple sites, enter a prefix that will be used for ticketing to determine which site the ticket is for.

Primary Site

Place a check in this box to set it as the default site to be used when creating tickets.

Address 1

Street number and name.

Address 2

PO Box or any other additional address related information.

City

City where site is located.

State

State where site is located

Postal Code

Postal Code where site is located.

Country

Country where site is located.

Email

Site's main contact email address or company email address.

Phone #

Site's main Phone #

Fax #

Site's Fax #

Your Logo

Using the browse button, user can select the Company logo to show on Reports.

Current Logo

Shows current logo selected.






Edit a Site




To Edit a Site, click on the Edit link to the right for the Site you would like to make changes to.



Any field can be edited in the menu. Note: Any tickets created prior to the Ticket Prefix being changed will not get updated with the new prefix.

When finished editing, click the 'Save' button, or 'Back to List'  if you do not want to save the changes you made.




Delete a Site



Click the Delete link to the right of the Site you would like removed. 

By default, Fulcrum has one site already in place. Note: You cannot delete ALL sites.




You will be prompted to confirm the deletion. Reaffirm by clicking the 'Delete' button. 

You will then be returned to the main menu and the site has been removed.
