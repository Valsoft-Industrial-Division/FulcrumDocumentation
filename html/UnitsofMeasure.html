Units of Measure






Units of Measure (UOM) is used to define measurement conversions that can be applied to a specific Product in Ticket Details when creating a ticket.

Fulcrum (by default) comes with 12 pre-configured UOM conversions (outlined in red).








Create a new UOM



Click the button to add a new UOM.

When finished, click the button. The new UOM will now show on the User UOM main screen.


Convert From

Adds a signature box on the ticket screen.

Convert To 

Select location (based on locations set up in Main Menu).

Description 

Give your new conversion a name.

Type

Weight - When the product is weighted (i.e Metric Ton).

Other -    A non weighted measure (i.e Yard).

Operation

Select operation to achieve the conversion. (Multiple or Divide).

Conversion

The number the Operation with Multiple or Divide from.


Edit a UOM



To Edit a UOM, click on the Edit link to the right for the UOM you would like to make changes to. 

Users can only edit the Description of a default UOM.



 

Users will be able to modified any field on a custom UOM. 

When finished editing, click the button, or if you do not want to save the changes you made.




Delete a UOM


To delete a custom UOM, click on the Delete link to the right for the custom UOM you would like to remove.




You will be prompted to confirm the deletion. Reaffirm by clicking the button. 

You will then be returned to the main UOM screen and the custom UOM has been removed.
