Customer Ticket List






The Ticket List menu shows all tickets that have been created and displays ticket details.


Number 

Displays ticket number (click to view ticket details), print icon (click to reprint ticket) and @ (to resend emailed ticket to customer).

Prefix

The ticket prefix set in Site Info if multiple sites exist. If a user has a specific site assigned to them, they will only be able to see tickets for that site.

Customer

The Customer the ticket is associated with.

Status

Shows the status of the ticket (Open, Closed, Void, Dispatch)

Remarks

Displays any remarks entered when ticket was created.

Created On

Displays the date and time the ticket was created on.





User's have the ability to filter tickets based on their status. To do this, click on the Show Ticket Status drop down box and select the status you'd like to show tickets for. 

Only tickets with the status you selected in the drop down will be displayed in the list. 







Create a Ticket



When creating a ticket, there are a few mandetory fields that need to be filled in (Vehicle, Customer, Price Sheet) in order to add ticket details. 

You can click on the button on select fields for quick lookup, or enter the % symbol to show all applicable records in for a field. 

If you only have one Site setup, the Site field will auto fill with that site. 



Number

Default number is 0. If left at 0, Fulcrum will auto increment the ticket number based on the last ticket entered.

Date (auto filled)

Shows current date.

Time (auto filled)

Shows current time.

Status

Select Ticket Receipt or Ticket Form. 

Site

Select the site the current ticket is for.

Staged Ticket (optional)

Enter a pre-configure ticket staging to allow for faster ticket creation. *Note: Field will only show if you have active Staged Tickets setup.

Hauler (optional)

Select a Hauler for your ticket. *Note: Field will only show if you have active Haulers setup.

Zone 

Select a Zone for your ticket. *Note: Field will only show if you have an active Haulers setup.

Vehicle

Select a Vehicle for your ticket.

Trailer

Select a Trailer for your ticket.

Customer

Select a Customer for your ticket.

Sales Order (optional)

Enter a valid Sales Order for the selected customer.

Price Sheet

Select a valid price sheet to calculate products rates and unit of measure. 

Remarks (optional)

Lets the customer input additional information.

Signature (optional)

Allows capture of a Signature on tickets.




Add Ticket Details



By default, you will see one ticket detail line. Click the Edit Details button to the right to open the ticket details window. 

Users can add multiple ticket detail lines if required.





Ticket Detail window



 



Product

Select the product for your ticket detail. *Note: the product list is limited to products from your selected Price Sheet.

Location (optional)

Used if you want to add location info to your ticket, or if you have multiple locations.

Invt. Container (optional)

If you're using the inventory feature, select the container you are depleting from. *Note: Field will only show if you have Inventory configured.

Direction

Select if this product is coming to your site, or if it is going out.

UOM

The unit of measure your scale is weighing in. Note: the Ticket Details UOM will automatically be set to the Default UOM in your site setup.

Gross

Records the Gross weight for ticket. Weight can be captured from the scale by clicking the scale icon to the right, or by manually entering it.

Tare

Records the Tare weight for ticket. Weight can be captured from the scale by clicking the scale icon to the right, or by manually entering it.

Adjustment

Used to adjust weight after weight has been recorded. *Note: Weight can be adjusted after ticket has been stored but ticket must have a status of open. 

Net 

Auto calculated once a gross and tare have been entered.  

Converted Qty.

The calculated conversion of your ticket uom to your rate uom (i.e pounds to tons).

Quantity (optional)

If the ticket product is measure per item (EA), enter how many pieces are coming in/going out.





If you need to enter additional ticket detail, click on the icon to add another detail line. 




The new ticket detail has been added to the ticket. Click the Edit Details button to the right to enter ticket details.






Delete Ticket Details



If you would like to delete a ticket detail line, click the delete icon to the right of the ticket detail.



When you delete a ticket detail, you will not get a confirmation message. The page will refresh and the detail line has been removed. 






Edit a Ticket Detail Rate



You have the ability to change the original rates set in your selected Price Sheet.

You can adjust a rate by clicking the button on the right side of the ticket detail line. 





You can update any rate within in the rate detail screen. You can also change if the rate is taxable or not and also change the tax code. 

Rate changes are applied as soon as the check box icon is clicked. 

*Note: You cannot update a product rate once a ticket has been saved (even if the ticket has a status of open).


Rate

Displays set rates from the applicable Price Sheet.  

UOM

The unit of measure (by uom, quantity or flat rate) being used for the rate.

Taxable

Toggle to set if a rate is taxable or not. 

Tax Code

Allows you to select your tax configuration that determines tax percentages.

Rate UOM

Sets the uom your rate is converted to.






Ticket Payment(s)


Once the ticket details are filled in, you can see the Pay Details for your ticket by clicking the button at the bottom of the ticket screen.





The Payment screen will show Product, Rate Type, Rate, Quantity, Sub Total, Tax Total and Line Totals for all detail lines on the ticket.

Users can select the payment type and the amount being paid on the ticket.





There are four Payment options available; Cash, Check, Credit Card and Escrow.




Cash Payment



Select 'Cash' under Payment Type and enter the amount to be paid if the customer is making a cash payment.





Check Payment



If a Customer is paying by check, select Check from the Payment Type drop down. Enter a check number, amount being paid and a

ACH/ECP (if applicable).





Credit Card Payment


 Select 'Credit Card' under Payment Type and enter the amount being paid.

You can select a card already on file using the drop down arrow, or add a new Credit Card by clicking the 'Add new Credit Card' button.

*Note: To use the credit card payment feature you will need to be signed up with BlueSnap




To add a new credit card, click the  'Add new Credit Card' button and enter the new credit card details. 

When finished, click the 'Create Vaulted Credit Card' button to return to the ticket screen. The new card will now be available for selection.

Credit Card setup can also be found in the Customer menu.






Escrow Payment


Select 'Escrow' under Payment Type and enter the amount being paid. Current Escrow balance for the customer is highlighted in green.

If there is not enough remaining in Escrow to pay the full ticket balance, you will be able to chose a second payment type for the remaining balance.

Saving Tickets


If this is your first pass, click 'Store and List'  to save the ticket and return to the main Ticket List. 

If you would like to create another ticket without returning to the main ticket list, click the 'Store and New' button. 



Print a Ticket


There is a Print Ticket box at the bottom just above the Store ticket options. Place a check in the Print ticket box and enter the number of copies you would like to print. 

*Note: Printers will need to be setup in Device Hub to print from the ticket screen.

If the 'Email Ticket' option is turned on in the customer's setup, a pdf copy of the ticket is automatically emailed to them when the ticket has been completed.

If you don't have a printer setup, you can click on the icon for the ticket you would like to print to open the pdf ticket in another browser window. 



The pdf ticket will open in a new browser tab.

On the top of the page you can see the page count, zoom in/out, fit to page and rotate the page.

At the top right of your screen you can download a pdf version of the ticket, or print the ticket to your windows printer.




Email a Ticket



Click the @ icon to the right of the Ticket you would like to send (ticket will be sent to email address configured in the Customer setup.





You will see a message box stating that the ticket was sent successfully. 




The Customer with receive an email with a PDF copy of the ticket attached.
