User Fields





User Defined Fields allows users to add additional functionality and design in Fulcrum.  

You can add custom fields with specialized rules for gathering data not supplied by default. 

User Defined Fields will appear on the ticket entry screen and provide custom details specific to your needs.







Create User Field


You can create customized fields for existing tables that can add specialized functionality and rules for gathering data.


Table Name

Choose the table you would like to add the new field to from the drop down menu. 

Name

Enter the name you would like for your field.

UI Name

Enter the text to display on screen for the field.

Data Type

Street number and name.

Required Entry

If checked will become a mandatory field.

Place Holder

Sets the control’s Place Holder (Water Mark) text

Regular Expression for Validation

Enter expression for validation.

Regular Expression Failure Message

Message to display when entry does not match pattern. 

Default Value

Enter a preset value that will show by default.






Edit User Field


Highlight the field you would like to edit and click on the 'Edit' button to the right.





Users can edit the UI Name, Place Holder, Regular Expression for Validation, Regular Expression Failure Message and the Default value for a table field.







Delete User Field

To delete a field, highlight the field you would like to remove and click on the button to the right.






User will be asked to confirm deletion. If you do not wish to continue with the field deletion, click and the field will not be removed. 

Clicking the button will remove the field and return you back to the list.
