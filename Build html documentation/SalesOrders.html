Sales Orders







The Sales Orders setup can be accessed from the Maintenance menu in Fulcrum.

Sales Orders are customer based and designated to one Price Sheet.




Create a new Sales Order



From the main screen, click the 'Create New' button to add a new Sales Order.


Customer

Enter the customer you would like to create the Order for.
Price Sheet

Used to add rates to a Sales Order that is used in the ticketing process to calculate a Product's cost.
Description 

Enter a description for your Vehicle.
Inactive

Mark if a Sales Order is Active or Inactive.
External ID 

Can to used to add a unique value or make reference to data you are importing from another system.
Print Pricing

Lets you chose if you would like the Pricing details to show up on printed tickets using this Sales Order.
Tax Exempt 

This will override any tax rates set on the Price Sheet for the Products on your Sales Order.
Address 

Address detail for Sales Order.
City

Address detail for Sales Order.
State

Address detail for Sales Order.
Postal Code

Address detail for Sales Order.
Country

Address detail for Sales Order.
Notes

User can add additional notes to provide more details to the Sales Order.
Products

Add preferred Products you would like to include in your Sales Order.






Add Product to Sales Order



 To add a Product to your Sales Order, click the button to the right.

 Enter a Product and select a direction for the Product.

 When you're finished, click 'Save' or click if you would like to add another product.






Add All Products



Click the 'Add all Products' button on the bottom of the Sales Order screen to add all active Products to your Sales Order.

When you're finished, click 'Save' or click if you would like to add another product.





Add Products from Category



Click the 'Add products from category' button on the Sales Order. 




A window will open and allow you to select a Product Class. Enter the Product you would like to add and click the 'Add Products' button.

When you're finished, click 'Save' or click if you would like to add another product.




Edit a Sales Order



To edit a Sales Order, click the "Edit' link to the right of the product you want to edit.



You can edit any field (except Customer) on the Sales Order.

When you're finished, click 'Save' to apply any changes.




Duplicate a Sales Order



You can create a copy of an existing Sales Order by clicking 'Duplicate' on the right side of the Sales Order you would like to copy.





 Update any fields on the Sales Order you would like to change (the name cannot be exactly the same as the Sales Order you duplicated).

When you're finished, click 'Save' to apply any changes.





Delete a Sales Order



 To delete a Sales Order, click the 'Delete' button to the right of the Sales Order you would like to remove. 




Confirm you would like to delete the Sales Order by clicking the 'Delete' button.




*Note: You cannot delete a Sales Order if it has been used on a ticket.
