Container Types






Container Types can be managed from the Inventory Menu.





Create a Container Type



There are two ways you can create a container type:


1. Inventory Menu


You can create a new container type directly from the Inventory menu by clicking the New Container Type button located at the bottom of the Container Types box.



2. Container Type Index



In the Container Types box, click on the Container Types button to open the Container Type Index.



Click the Create New button to the right of the screen to add a new record. 




On the create new screen, give your Container Type a description.

The External ID field (optional) can be used as a unique identifier specific to your needs.

When you are finished entering your data, click the Create button to add the container type, or Back to List if you would like to discard the record.




Edit a Container Type



On the Container Type Index screen, click Edit to the right side of the screen for the container type you would like to edit.

When you are finished editing, click the Save button to update the container type, or Back to List if you would like to discard the changes made.






Delete a Container Type


From the Container Type Index screen, click Delete to the right side of the container type you would like to remove.



A confirmation screen will appear asking you to confirm the deletion.

To confirm to record removal, click the Delete button. If you would like to keep the record, click the Back to List button and return to the main menu.
