Customers






The Customer setup can be accessed from the Main Menu of the Home screen. 

User's will have the option to Create a new Customer and Edit, Pay Tickets or delete a current Customer. 







Create New Customer




Click the button to add a new Customer.


Description

Set a description (name) for your Customer.

External ID

Can to used to add a unique value or make reference to data you are importing from another system.
Inactive
Mark if the staged ticket is Active or Inactive.

Address 1 

Customer's current address information.

Address 2

Customer's current address information.

City

City where Customer is located.

State

State where Customer is located.

Postal Code

Postal Code for Customer's address.

Country

Country where Customer is located.

Phone #

Customer's main Phone #

Mobile #

Cell phone # for customer.

Fax #

Customer's Fax #

Email

Customer's main contact email address.

Email Tickets

If checked, any ticket created for this customer will be sent to the email address specified above.

Ticket Form

Select the ticket form you would like use based on individual customers (if left blank, it will default to the ticket form set in Manage Settings).

Ticket Copies

Allows you to set the number of ticket copies printed on a per customer basis. If set to 0, it will print the set # of copies in Device Hub.

Require payment for tickets

All tickets for this Customer must be paid before the ticket can be closed.

Escrow Minimum Balance 

The minimum amount that can be paid on tickets from Escrow.

Notes

Add additional notes to customer profile

Custom Fields

Any custom fields created through the Admin menu for the customer table will show at the bottom portion when creating customers.







Edit a Customer


From the Edit Customer screen, you will be able to modified any field.

To Edit a Customer, click on the Edit link to the right for the Customer you would like to make changes to. 



All fields are editable from this screen. 

When finished editing, click or if you do not want to save the changes you made. 






Add a Customer Credit Card


From the Edit Customer screen, click the 'Vaulted Credit Cards' button at the bottom of the screen.





On the Vaulted Shoppers screen, click the button.





Enter the Credit Card details and click the button.






The new Credit Card will now show on the Vaulted Shoppers screen for the Customer, and will show in the the drop down box when you create a ticket for this Customer.







Pay Tickets 



If a Customer would like to pay multiple tickets at once or Pay Cash, Check, Credit Card, or Escrow once a week, day, month or any period of time.

Tickets that are marked with a status of "Hold for Payment" can later be accessed for payment from the Customer menu by clicking the button.



On the Pay Tickets screen, place a check in the 'Pay?' box of each ticket you would like to pay on and then select a Payment Type. 





Payment Options


There are four Payment options available; Cash, Check, Credit Card and Escrow.

Only one form of payment is allowed per ticket and it must cover the total amount of all tickets selected. 




Cash Payment


Select 'Cash' from the Payment Type drop down list and click the 'Pay' button. 







Check Payment


Select 'Check' from the Payment Type drop down list.

Enter a check number (optional) and then click the 'Pay' button.





Credit Card Payment


Select 'Credit Card' from the Payment Type drop down list.

You can select a card already on file using the drop down arrow, or add a new Credit Card by clicking the 'Add new Credit Card' button.





Escrow Payment



Select 'Escrow' from the Payment Type drop down list.

If the Customer has money in Escrow on their account, the total ticket amount will be deducted from their Escrow balance. 






Delete Customer


To Delete a Customer, click on the 'Delete' link to the right of the Customer you would like to remove.



A prompt will show to confirm the deletion. Reaffirm by clicking the button. 

You will then be returned to the main Customer List screen and the Customer has been removed.
