CustomerTicket Staging






Ticket Staging can speed up the process of ticketing if you have a field that is frequently used.

All fields are optional. You can set as many (or as little) as you like. 


Description

Give your staged ticket a name.

External ID

Can to used to add a unique value or make reference to data you are importing from another system.
Inactive
Mark if the staged ticket is Active or Inactive.

Hauler 

Select a Hauler for your ticket.

Haul Zone

Select a Zone for your ticket.

Vehicle

Select a Vehicle for your ticket.

Trailer

Select a Trailer for your ticket.

Customer

Select a Customer for your ticket.

Sales Order

Enter a valid Sales Order to use for the Staged Ticket.

Price Sheet

Enter a valid Price Sheet to use for the Staged Ticket.

Product

Select a Product for your ticket.

Remarks

Allows you to add a custom note.

Use Count

Set the amount of times the staged ticket can be used.

Total Uses

Displays the amount of times the staged ticket has been used in ticketing. 





Edit/Delete a Staged Ticket


To Edit/Delete a staged ticket, click on the Edit or Delete link to the right for the staged ticket you would like to make changes to or delete.

Editing - Users will be able to modified any field when editing.

When finished editing, click 'Save' or 'Back to List if you do not want to save the changes you made.

Deleting - User will be prompted for confirmation to remove the record.




Using your Staged Ticket


After your Staged Ticket has been made, create a ticket and enter the staged ticket name. 



The ticket is then populated with the fields you set on the staged ticket. 




Set a Use Count for Staged Ticket


Users can specify how many times a Staged Ticket can be used.  

If the Use Count is set to 0 the staged ticket use is unlimited (staged ticket will stay active until manually set as inactive). 

When the 'Total Uses' count has been reached for the staged ticket, the staged ticket will automatically be set to Inactive.

User's can create new staged tickets (or edit the use count on the existing staged ticket) and reactivate the record to make it available for use again.
