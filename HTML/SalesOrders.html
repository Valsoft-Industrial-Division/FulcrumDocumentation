Sales Orders






  • The Sales Orders setup can be accessed from the Main Menu of the Home screen.
  • This feature allows you to define product rates for a Sales Order specific to selected customer.
  • These defined rate values will automatically appear on a Customer's ticket when that Customer's Sales Order and material are selected at time of ticketing. 








Create a new Sales Order



  • From the main screen, click the button to add a new Sales Order.


Customer 

Enter the customer you would like to create the Order for.

Price Sheet 

Used to add rates to a Sales Order that is used in the ticketing process to calculate a Product's cost.
Description 
Enter a description for your Vehicle.
External ID 
Can to used to add a unique value or make reference to data you are importing from another system.
Inactive 
Mark if a Sales Order is Active or Inactive.

Address 1 

Site's current address information.

Address 2 

Site's current address information.

City 

City where site is located.

State 

State where site is located.

Postal Code 

Postal Code where site is located.

Country 

Country where site is located.

Start Date 

The date you would like your Sales Order to begin.

End Date 

The date you would like your Order to Expire. After the expiry date has passed, the Sales Order will no longer show for selection when creating a ticket.

Notes 

User can add additional notes to provide more details to the Sales Order.







Add a Product



  • To add a Product to your Sales Order, click the to the right.
  • When the new line appears, enter the Product you would like to add and select a direction.








Add All Products



  • Click the button on the bottom of the Sales Order screen to add all active Products to your Sales Order.
  • Click to add all products and return to the main menu.








Add Products from Category



  • To pick a Product to add to your Sales Order, click the button on the bottom of the Sales Order screen.
  • A window will open and allow you to select a Product Class.
  • Enter the Product you would like to add and click the button.





  • You will be returned to the Sales Order main screen and all products from the product class will now show in the Sales Order details.
  • If you are finished entering your details, click to be returned to the main menu.
