Vehicles








The Vehicle setup can be accessed from the Main Menu of the Home screen. 

User's will have the option to Create, Edit or Delete a record from this screen.






Create new Vehicle




Click the button to add a new Vehicle.


Description - Enter a description for your Vehicle.
External ID - Can to used to add a unique value or make reference to data you are importing from another system.
Inactive - Mark if a Vehicle as Active or Inactive.





Edit Vehicle




To Edit a Vehicle, click on the Edit link to the right for the Vehicle you would like to make changes to.



 From the Edit Vehicle screen, you will be able to modified any field. 

When finished editing, click or Cancel if you do not want to save the changes you made. 





Delete Vehicle


To Delete a Vehicle, click on the Delete button to the right for the Vehicle you would like to remove.




You will be prompted to confirm the deletion. Reaffirm by clicking the 'Delete' button. 

You will then be returned to the main Vehicle List screen and the Vehicle has been removed.
