Permission Sets





Permission Sets allow you to control what areas of Fulcrum users will have access to.

You can remove permission sets, create new ones and update ones already in place. 






Create Permission Set



Set a name for your new permission set and click on .

You will be returned to the main Permission Set screen and your new Permission Set will show in the list.







Update Permission Set



From here, you can click on the Update button to edit your Permission Set. 






Controlling User access to features



If you do not want a users to have access to specific areas of Fulcrum, you can turn on the No Access feature for the menu you would like to block access to. Note: If No Access is turned on, it will override all other options for that menu.

When you're finished editing, click the Save button, or Back to List if you do not want to save the changes you made.

Once you've finished configuring the permissions, confirm the user you want to limit is assigned the correct permission set under Manage Users.







If a User assigned to that Permission Set tries to access the area, they will receive a warning message.








Delete Permission Set


To delete a permission set, click the Delete button on the permission set you would like to remove. 





You will be prompted to confirm the deletion. Reaffirm by clicking the Delete button. 

You will then be returned to the main permissions set screen and the permission set has been removed.
