Containers







From the Inventory Menu, you can create, edit or delete inventory containers.







Create a new Container




There are two ways you can create a container:


1. Inventory Menu


You can create a new container directly from the Inventory menu by clicking the New Container button located at the bottom of the Container box.





On the create new Inventory Container screen, give your Container a description.

The External ID field (optional) can be used as a unique identifier specific to your needs.

When you are finished entering your data, click the Create button to add the container, or Back to List if you would like to discard the record.





2. Container Index



In the Containers box, click on the Containers button to open the Inventory Container List.





Click the Create New button to the right of the screen to add a new record. 




On the create new Inventory Container screen, give your Container a description.

The External ID field (optional) can be used as a unique identifier specific to your needs.

When you are finished entering your data, click the Create button to add the container, or Back to List if you would like to discard the record.






Edit a Container



From the Inventory Container List screen, click Edit to the right for the container you would like to edit.




Grayed out fields cannot be edited. If the data you entered in a grayed out field isn't correct, you will have to delete the record and create a new one with the correct information.  

When you are finished editing, click the Save button to update the record, or Back to List if you would like to discard the changes made.








Adjust Container Inventory



You can adjust inventory (add or remove) on an existing container by clicking the Adjust Inv. link to the right of the container you would like to adjust. 





Enter an adjustment note (which will be used as the description in your Inventory Journal) and the amount of inventory you would like to adjust. 

If you are removing inventory, use the minus sign (i.e -500) in front of the number. 

When you are finished entering your data, click the Create button to complete the adjustment, or Back to List if you would like to discard the adjustment.




Your adjustment will be added to the Inventory Journal, showing your previous amount, adjustment amount and calculate the current amount of inventory in the container. 







Delete a Container


From the Inventory Container List screen, click Delete to the right for the container you would like to remove.



A confirmation screen will appear asking you to confirm the deletion.

To confirm to record removal, click the Delete button. If you would like to keep the record, click the Back to List button and return to the main menu.
